All in One SEO Pack Pro v4.5.3.1 汉化版

2024-10-21

插件介绍

All in One SEO Pack Pro：是用于搜索引擎优化的流行插件之一，在 WordPress 上有超过 2 百万个活动安装。该插件的控制面板分为三个主要部分-常规设置，XML Sitemap 和功能管理器，最新版本中加入了 ChatGPT 支持，是一款非常优秀的 SEO 插件

All in One SEO Pack Pro 核心功能：

  • 轻巧简洁的代码
  • Google AMP SEO 选项
  • 自动生成 XML 网站地图
  • 预览片段
  • 每个帖子或页面更新后对搜索引擎执行 Ping 操作
  • Robots.txt 编辑器
  • 视频站点地图支持
  • 图像 SEO 设置
  • .htaccess 文件的文件编辑器
  • 阻止行为不端的机器人
  • 进出口商
  • 社交元模块
  • WooCommerce SEO 设置
  • 与网站管理员工具连接的选项
  • 模式支持
  • 54 种语言翻译支持
  • 分类和标签的 SEO 选项
  • Google 跟踪代码管理器支持
  • Google 知识图支持

All in One SEO Pack Pro v4.5.3.1 更新日志

  • 修复：当用户执行不相关的操作时，AIOSEO 设置模式有时会在 Divi 可视化编辑器中打开。
  • 新增内容：添加了对将位置从 Yoast SEO 导入本地 SEO 的支持。
  • 新增内容：在重定向管理器中添加了对 LiteSpeed 服务器的支持。
  • 更新：大量性能更新。
  • 更新：SEO 修订现在跟踪基石内容设置的更改。
  • 更新：重定向的手动 404 现在加载 404 WP 模板，而不仅仅是向浏览器发送 404 标头。
  • 更新：在链接助手的链接报告中添加了帖子状态徽章，以显示未发布帖子的帖子状态。
  • 更新：改进了 AIOSEO 和 Extra 主题之间的样式兼容性。
  • 已修复：分页格式现在使用全局分隔符标记。
  • 已修复：当产品没有有效的价格设置时，不再输出产品架构。
  • 已修复：各种模式图现在在输出日期时考虑时区，并且不再输出不正确的时间戳。
  • 已修复：当帖子被删除时，重定向模式 UI 现在可以正常工作。
  • 已修复：重定向不再阻止使用 WP CLI。
  • 已修复：在字段中插入值后，重定向的 HTTP 标头表不再消失。
  • 已修复：编辑重定向时，选择值后编辑屏幕不再关闭。
  • 已修复：导入具有某些正则表达式模式的 robots.txt 规则时出错。
  • 已修复：导致某些本地化字符串被排除在语言文件中的文件结构问题。
  • 已修复：删除了附件页面的基石内容设置。
  • 已修复：设置向导有时会显示错误的步数。
  • 修复：未选择主要类别时，片段预览显示分类标题智能标记的预览不正确。
  • 已修复：“搜索统计”现在可以正确地将主页与“设置”>“阅读”下选择的内容同步。
  • 已修复：插件与自动优化插件冲突，当登录用户使用俄语作为其区域设置时，网站前端显示白屏。
  • 已修复：与 WP 6.0+ 中基于块的主题冲突。
