LESLIE Note v5.33 绿色便携版

2024-09-11

LESLIE Note 是由个人开发的一款本地笔记软件，外观借鉴了 OneNote 软件，软件是一款纯本地的笔记软件，界面做的还不错，内容完全本地，原生便携，是一款轻量的免费绿色软件。

软件特点

1，笔记簿 ——1.1，LN 的基本存储单位是“笔记簿”，“笔记”存放于“笔记簿”中。 ——1.2，LN 支持添加任意多个“笔记簿”。 ——1.3，“笔记簿”内支持添加任意多篇“笔记”。 ——1.4，“笔记”支持添加“附件”。 2，附件 ——2.1，数量不限。 ——2.2，格式不限。 ——2.3，自动加密。 3，加密 ——3.1，“笔记簿”自动加密，无法从外部获取笔记内容。 ——3.2，“笔记”支持设置密码，即“加密笔记”，查看/修改/导出等操作时需要输入正确的密码后才可操作。 ——3.3，“附件”自动加密。 ——说明 1，加密是根据用户设置的“笔记簿”密码来生成加密的，所以每个人、每个“笔记簿”的加密都不一样。 ——说明 2，LN 不会保存“笔记簿”密码、“加密笔记”密码等密码。LN 只保存了一个将密码 MD5 加密后的字符串，用于校验输入的密码是否正确。（MD5 是不可逆的，即：无法通过 MD5 加密后的字符串反计算出原密码。） 4，历史版本 ——4.1，每次保存自动生成历史版本副本。 ——4.2，支持无限次历史版本。 5，无限层级树状分类 6，回收站 7，批量导出 导出内容： ——7.1，笔记本身，笔记格式：.txt/.rtf/其它（由“编辑器”决定）。 ——7.2，附件。 ——7.3，附属信息：类别、级别、标签、摘要等。 8，批量导入 导入方式： ——8.1，导入由 LN 导出的笔记。在导入时除笔记和附件外，会自动为添加上附属信息（类别、级别、标签、摘要等）。 ——8.2，导入本地.txt/.rtf 文件。支持自动添加附件：只需要将附件件放入到同目录下与.txt/.rtf 同名的文件夹内即可。 ——8.3，导入其它格式笔记。此操作需要“编辑器”本身支持。 9，备份 ——9.1，一键备份当前“笔记簿”。 ——9.2，一键备份所有“笔记簿”。 ——说明：备份时自动打包为.zip 压缩包，打包时支持打包为加密压缩包。 10，编辑器 ——支持“编辑器”插件，除了自带的纯文本编辑器、富文本编辑器外，可以自行添加多种编辑器，以实现更加专业细致的功能。

更新日志

1，增加：《LN Mobile》（LESLIE NOTE 移动版）同步支持。 2，增加：“自动更新摘要”【综合设置[通用]】设置项。 3，增加：左下信息框增加当前笔记簿占用体积信息。 4，优化：插件编辑器管理工具，简化操作界面。

相关说明

1，本程序是绿色版，使用时请直接运行‘LESLIE NOTE.exe’即可。也可以创建快捷方式到桌面。 2，环境需求： 2.1，.Net Framework 4.5 或以上版本。 2.2，VC++ 2010 运行库或以上版本。
