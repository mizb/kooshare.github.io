Drivvo Pro v8.5.1 for Android 解锁专业版

2024-08-31

Drivvo Pro 是一款监控汽车的健康状况，并控制支出、收入费用的车辆管理应用。使用 Drivvo Pro 可以控制加油，维修保养，生成报表并为您的车辆的重要事件设定提醒。

Drivvo Pro 不仅可以管理汽车的燃料消耗和经济，而且可以管理摩托车，公共汽车或卡车的维护费用和服务费用。而且您还可以计划您的下一个供应。在哪里加油是一项协作功能，可以检查您附近和您喜欢的其他位置的加油站的燃油价格。您仍然可以请求将GPS引导到所选站点。

精彩功能：

•总里程表读数或行程里程输入 •全面的燃料管理（燃料成本，效率，加油站等），以及用于财务控制的加油类型（全部，部分）的高级选项 •支持记录加油-上门服务，费用，差旅费（商务，个人等）。 •具有多个单位支持的汽油里程（燃油经济性或燃油里程）计算：MPG，加仑/ 100mi，mi / L，km / gallon，L / 100km，km / L •加油站注册 •许多统计数据/图表/报告为您提供财政控制 •每个加油站/加油站的耗油量图表 •高级提醒（例如下次汽车保养的提醒） •图表按汽车和/或日期范围进行过滤。轻松编辑图表中的数据。 •带有提醒的车辆维护记录管理 •距离统计：总距离，距离/年，距离/月，距离/天 •成本统计：每个类别的成本（加油，账单，服务）。成本/距离，成本/年，成本/月，成本/天，消耗量等… •燃油效率统计和管理 •每个加油站/加油站的燃气效率统计 •燃油效率最佳/最差的报告。容易跟踪您所处的状况是好是坏。 •跟踪车辆的维护，服务，成本，费用 •燃油单位：美国加仑，英制加仑和升 •许多燃油效率单位，例如MPG和Km / L •跟踪加油量并节省现金 •您还可以从其他应用程序恢复数据。（Fuelio，aCar，Carango，加油站，我的汽车，汽车费用，加油站管理器） •加油站的位置在地图上显示为带有品牌的图钉，上面显示了最近的燃油价格。 •找到您附近价格最低的加油站（无论哪种类型）。 •报告汽油价格，以帮助其他人节省金钱并提高您所在社区的排名 •收入：考虑到那些将车辆用作工作工具的人，DRIVVO现在还可以记录收入。新功能是为那些专业使用车辆并需要实时了解与之相关的费用和收入的人而开发的。
