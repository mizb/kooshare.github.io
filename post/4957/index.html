不忘初心 Windows10 22H2 (19045.4957) x64 纯净 深度精简 无更新 (2024.09.26)

2024-09-29

注意此版不能更新补丁，支持人脸、指纹、可以安装商店、以及其他 UWP Apxx 程序，可以登录微软账号，此为深度精简版体积小、精简的比较多，注意不支持 bitlocker，适合软件不多的以及轻办公的用户使用，如有第三方软件打不开，请自行安装资源包里的微软常用运行库，为了保证稳定初心的系统全部都是离线精简和优化，非二次封装。系统纯净、流畅、进程少无任何第三方软件、欢迎朋友们测试反馈，提出意见！

更新记录

2024.9.26 更新记录 #修复 WIN+V 剪贴板 点击不粘贴的问题。 #更新安装包 MicrosoftEdgeX64_129.0.2792.52.msi

#KB5043313（19045.4957）微软官方更新如下：

  • [开始菜单] *新增功能！* 此更新会将个人资料图片移动到不同的位置。 菜单上的左窗格还有一个新的背景色。 此颜色更改使窗格上的项更加醒目。 此更改可能并非对所有用户都可用，因为它将逐步推出。
  • [IE 模式] 使用 IE 模式时，Microsoft Edge 有时会停止响应。
  • [媒体播放] 使用某些环绕声技术时，某些媒体的播放可能会停止。
  • [Microsoft Entra 单一登录 (SSO) ] SSO 请注意， (DMA) 的《欧洲数字市场法案》要求提示过于频繁。 使用证书进行身份验证时会发生这种情况。 若要了解详细信息，请参阅 即将对 Windows 单一登录进行的更改。
  • [国家/地区和操作员设置资产] 此更新使某些移动运营商的 COSA 配置文件处于最新状态。
  • [文件资源管理器和任务栏] 当你使用文件资源管理器和任务栏等应用时，Windows Server 会停止响应。
  • [Windows 更新选择加入通知] 登录时，它们现在会显示。
  • [目录枚举] 如果目录具有具有较长目标名称的符号链接，则此操作可能会失败。
  • [Microsoft Defender for Endpoint] 打开 Defender for Endpoint 时，工作文件夹文件无法同步。

系统特色

  • 此版不能正常更新补丁。

  • 纯净 无任何第三方软件。

  • 追求稳定、没有过度优化，可以正常办公。

  • 总共优化 150 多项系统内容，系统流畅更胜一筹。

  • 优化后台进程首次开机大约 50 个进程左右；

  • 禁用后台 40 多项系统不常用服务

  • 需要用到指纹的，请用自建账户，Admin 不支持;

  • 彻底阻止 IE 浏览器跳转 Micosoft Edge。

  • 任务栏时间显示秒数。

  • 修复精简后常见的 “A 字出现了问题,无法安装功能” 的问题。

  • 提升自建账户权限，自建账户将拥有和 “Administrator"管理员账号一样的权限（首次安装重启后生效）。

  • 避免以后升级到更高版本的系统，所有网络已经设置为按流量计费，检查更新后 下载补丁需经过你同意，但不影响系统给设备打驱动功能;

  • 防止自动更新补丁，设置 Windows Update 暂停更新到 2099 年 5 月 28 日。

  • 提供“禁止系统自动安装驱动.reg” ,不需要系统自动打驱动的 请安装系统后 立即导入即可;

  • 优化共享打印服务，避免局域网共享打印错误。

  • 系统恢复界面添加高级启动选项

  • 使用 Compact OS 释放空间，C 盘占用空间可减少 2G 左右

  • 电源模式新增 卓越模式

  • 高级电源设置 USB 选择性暂停 已禁用 避免有些 USB 设备断电

  • 禁用休眠文件，可减少 C 盘体积，如需开启 请使用管理员运行 CMD 命令：powercfg /hibernate on

  • 默认关闭 7G 预留空间;

  • 恢复要 “使用此计算机必须输入用户密码”的勾选框，原版给隐藏了;

  • IE 主页默认设置为百度，原版是先打开微软再跳转百度;
